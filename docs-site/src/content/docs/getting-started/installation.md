## Install the Extension

1. Open VS Code
2. Go to the Extensions view (`Ctrl+Shift+X`)
3. Search for **Copilot AI Flow**
4. Click **Install**

Or install from the command line:

```bash
code --install-extension feima.copilot-ai-flow
```

## Requirements

- VS Code 1.90+
- GitHub Copilot Chat extension installed and authenticated

## Verify Installation

Open the Copilot Chat panel (`Ctrl+Shift+I` on Windows/Linux, `Cmd+Shift+I` on Mac) and type:

```
@flow /list
```

You should see the built-in flow library. If not, reload VS Code (`Ctrl+Shift+P` → "Developer: Reload Window").
