## What Is Copilot AI Flow?

**Orchestrate multiple AI roles in a single command.** Define a team of specialized reviewers, planners, or investigators — each with their own perspective, tools, and context — and let them work together on your task.

## Quick Start

```bash
@flow /create a code review with separate security and style lenses
```

Then run it:

```bash
@flow #file:.github/flows/code-review.flow.yaml
```

That's it. Three specialized reviewers analyze your code in sequence and produce a prioritized verdict.
