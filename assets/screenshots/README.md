Screenshots used by the VS Code Marketplace listing. These are copies of the canonical assets in [`docs-site/public/assets/screenshots/`](../../docs-site/public/assets/screenshots/).

| File | Description |
|------|-------------|
| `code-review.png` | Code review flow output with prioritized findings |

## Tips

- Use **dark theme** — displays better on the Marketplace page
- Capture at **2× resolution** on a HiDPI display for crisp rendering
- Crop to the relevant UI — exclude OS chrome
- PNG format recommended
